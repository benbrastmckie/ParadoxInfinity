Hilbert’s Hotel

    The Hotel

    Hilbert’s Hotel is an infinite hotel: for each natural number 0, 1, 2, 3,… there is a room with that number. The hotel is completely full. Guest 0 is staying in room 0, Guest 1 is staying in room 1, and so forth.

    When a finite hotel is completely full, there is no way of accommodating additional guests (unless guests are willing to share rooms, which our guests are not). In an infinite hotel, however, the fact that the hotel is full is not an impediment to accommodating extra people.

    Say that Onyeka is a new guest. All the hotel manager needs to do to find her a room is make the following announcement: “Would guest \(n\) kindly relocate to room \(n+1\) ”. If all current guests abide by this request, Guest 0 will end up in room 1, Guest 1 will end up in room 2, and so forth. So even though all of the original guests still have a room of their own under the new arrangement, room 0 is available for Onyeka!

    It doesn’t stop there. Suppose 5 billion new guests arrive to our full hotel. We can accommodate them too! This time our announcement is this: “Would guest \(n\) kindly relocate to room \(n + 5 \text{ billion}\)”. In a similar way, we can accomodate any finite number of new guests!

    We can, in fact, even accommodate infinitely many new guests. If infinitely many guests arrive, we can tell each of the current guests “if you are in room \(n\), please relocate to room \(2n\) ”. That will leave every odd numbered room free for our infinitely many new guests to occupy.


    A Paradox

    Consider the set of original guests, which we’ll call “\(G\)”, and the set of original guests plus Onyeka, which we’ll call “\(G^+\)". Here is an argument.

    1. \(G^+\) has everything \(G\) has in it, plus one more thing.
    2. Conclusion: \(G^+\) has more things in it than \(G\).

    That looks like a pretty good argument. Usually, if you have some apples, and somebody gives you an extra apple, you then have more apples. Seems like similar reasoning should work for hotel guests.

    Here is another argument.

    1. There are as many guests in \(G\) as hotel rooms in Hilbert’s Hotel.
    2. There are as many guests in \(G^+\) as hotel rooms in Hilbert’s Hotel.
    3. \(G^+\) has the same number of things in it as \(G\).

    That also looks like a pretty good argument. The first two premises definitely look right — there is one guest in \(G\) per hotel room, and there is also one guest in \(G^+\) per hotel room. And surely, the conclusion follows from those two premises, no?

    But if both these arguments work, then \(G^+\) both has more things in it than \(G\) and has the same number of things in it as \(G\). That can’t be right! Paradox.

Cardinality

For a long time, people thought that paradoxes like the one above showed that there was something wrong with trying to use notions like “same number as” or “more than” when we’re talking about infinite sets. But then came Georg Cantor, 19th Century German mathematician, who had the essential insight, when it comes to infinity. He saw that there was a way of making precise notions like “more than”, “less than” and “same number as”; that, if you made these notions precise in this way, then paradoxes like the one above were resolved; and that these new, precise notions were extremely fruitful, mathematically.

Let me explain.

    Bijections; Injections; Surjections

    Here are the key notions. Suppose you have two sets: \(A\), the set of apples, and \(B\) the set of bananas.

    • There is an injection from \(A\) to \(B\) iff you can pair every apple with a distinct banana (and possibly have some bananas left over).

    [More formally: a function \(f\) is an injection from \(A\) to \(B\) iff, for all \(a_1, a_2\in A\), if \(a_1\neq a_2\), then \(f(a_1)\neq(f(a_2)\) (and \(f(a_1), f(a_2)\in B\)).]

    • There is a surjection from \(A\) to \(B\) iff you can pair every apple with a banana, and not have any bananas left over (though maybe different apples get paired with the same banana).

    [More formally: a function \(f\) is a surjection from \(A\) to \(B\) iff, for all \(b\in B\), there exists an \(a\in A\) such that \(f\(a\)=b\).]

    • There is a bijection from \(A\) to \(B\) iff you can pair every apple with a distinct banana, and not have any bananas left over — i.e., there is an injection from \(A\) to \(B\) that is also a surjection.

    [Slightly more formally: \(f\) is a bijection from \(A\) to \(B\) iff \(f\) is an injection from \(A\) to \(B\) and \(f\) is a surjection from \(A\) to \(B\).]


    Extending the Notion of Number

    Cantor’s insight, and what he helped prove, is that a perfectly coherent and extremely fruitful way of thinking of the notion of “same number as”, which works for both finite and infinite sets, is to just let “set \(A\) has the same number of members as set \(B\)” mean “there is a bijection from set \(A\) to set \(B\)”.

    Similarly, it is coherent and fruitful to let “the number of members of \(A\) is less than or equal to the number of member of \(B\)” mean “there is an injection from \(A\) to \(B\)”.

    That is a relatively obvious and simple insight, when it comes to finite sets. It accords with all the intuitive things you want to say about “same number as” and “less than or equal to”.

    But there is an important difference between finite sets and infinite sets. When we’ve got a finite number of apples, and a finite number of bananas, and there is a bijection between the apples and the bananas, there isn’t also a bijection from the apples to some proper sub-set of the bananas; it never happens that you can eat some of the bananas and still have the same number of apples as bananas.

    That’s not so when we’re talking about infinite sets. Take \(G\), the set of original guests at Hilbert’s Hotel, and \(G^+\), the set of original guests plus Onyeka. There is a bijection between \(G\) and a proper sub-set of \(G^+\) — it’s just the identity function, which pairs guest 0 with herself, guest 1 with himself, and so on. There is also a bijection between \(G\) and \(G^+\). You just pair Onyeka with guest 0, guest 0 with guest 1, guest 1 with guest 2,…

    So, in endorsing Cantor’s sense of “the same number as”, we must reject the principle that a set \(A\) cannot have the same number of things in it as a set \(B\) that is in bijection with a proper subset of \(A\). That is, we reject the first of the two argument above:

    1. \(G^+\) has everything \(G\) has in it, plus one more thing.
    2. \(G^+\) has more things in it than \(G\).

    That is not a valid argument, using Cantor’s sense of “more things”. On the other hand, this is a valid argument:

    1. There are as many guests in \(G\) as hotel rooms in Hilbert’s Hotel.
    2. There are as many guests in \(G^+\) as hotel rooms in Hilbert’s Hotel.
    3. \(G^+\) has the same number of things in it as \(G\).

    To be clear that we are talking about a more precise, and less intuitive, notion than the ordinary one, instead of talking about the “number of members” of a set, we will talk about the cardinality of a set.

    By definition,

    the cardinality of a set \(A\) equals the cardinality of a set \(B\) iff there is a bijection from \(A\) to \(B\).

    Similarly, by definition

    the cardinality of \(A\) is less than or equal to the cardinality of \(B\) iff there is an injection from \(A\) to \(B\).

    Here is some new notation: “\(|A|\)” means “the cardinality of \(A\)”. We can put the above definitions as follows: \[|A|=|B| \text{ iff there is a bijection from }A\text{ to }B\] and \[|A|\le|B| \text{ iff there is an injection from }A\text{ to }B\]

    These are the two most basic definitions in the theory of cardinalities. Everything else is defined in terms of these. So, for instance, \(|A|<|B|\) is defined as \(|A|\le|B|\) and \(|A|\neq|B|\). And \(|A|\ge|B|\) is defined as \(|B|\le|A|\).

    This is all equivalent to the normal, intuitive sense of “number”, when we are talking about finite sets; it extends that intuitive notion into the unfamiliar territory of infinite sets.


    Some Properties of Bijections

    Here are some things that are handy to know about bijections.

    1. Bijections are reflexive. That is: there is a bijection from every set \(A\) to itself. For example, the identity function \(f(a)=a\), for all \(a\in A\).

    2. Bijections are symmetric. That is: if there is a bijection from \(A\) to \(B\), there is a bijection from \(B\) to \(A\). (In particular, if \(f\) is a bijection from \(A\) to \(B\), \(f^{-1}\) is a bijection from \(B\) to \(A\). Convince yourself!)

    Because of this fact, we often say there is a bijection between \(A\) and \(B\), rather than saying there is a bijection from \(A\) to \(B\).

    1. Bijections are transitive. That is: if there is a bijection from \(A\) to \(B\), and a bijection from \(B\) to \(C\), there is a bijection from \(A\) to \(C\). (In particular, if \(f\) is a bijection from \(A\) to \(B\), and \(g\) is a bijection from \(B\) to \(C\), then \(g\circ f\) is a bijection from \(A\) to \(C\). Convince yourself!)

    Because bijections have these three properties, the definition we gave above of when two sets have the same cardinality is a good definition, and “=” is an appropriate symbol to use.


    Some Properties of Injections

    Here are some handy things to know about injections, and hence about the “less than or equal to” relation among cardinalities.

    1. Injections are reflexive (just like bijections, above).
    2. Injections are transitive (just like bijections, above).

    Here is the really interesting one:

    1. Injections are anti-symmetric. That is: if there is an injection from \(A\) to \(B\), and an injection from \(B\) to \(A\), then there is a bijection from \(A\) to \(B\).

    In other words: if \(|A|\le |B|\), and (\|B|\le |A|\), then \(|A|=|B|\)

    Properties (1) and (2) are pretty easy to prove. (Give it a go!) Property (3) is not trivial; it is most commonly called “the Schröder-Bernestein Theorem”, though it was first proved by Richard Dedekind. (Dedekind seems to have had a habit of proving things and not telling anyone about it, which is why his name doesn’t appear everywhere it should.)

    These three facts together mean that \(\le\), as defined for cardinalities, is a partial order, in just the way that you would expect a property that uses “\(le\)” as its symbol to be.

    For \(\le\) to be a total order (also called a “linear order”), the following also needs to be true:

    1. Injections are total. That is: for all sets \(A\) and \(B\), either there is an injection from \(A\) to \(B\) or there is an injection from \(B\) to \(A\).

    To prove this, you need to use the Axiom of Choice. That is an axiom of set theory which is usually assumed, these days, though it remains the most controversial axiom of standard set theory. We will have more to say about the Axiom of Choice later in this course.


    An Example

    Here are two sets that have the same cardinality: \(\mathbb{N}\), the set of natural numbers, and \(\mathbb{Z}\), the set of integers.

    Proof: let \(f(n)\) be the function such that

    • if \(n\) is an even natural number, \(f(n)=\frac{n}{2}\), and
    • if \(n\) is an odd natural number, \(f(n)=-\frac{n+1}{2}\)

    Thus:

    \(n\) \(f(n)\) another way of representing \(f(n)\)
    0 \(\frac{0}{2}\) \(0\)
    1 \(-\frac{1+1}{2}\) \(-1\)
    2 \(\frac{2}{2}\) \(1\)
    3 \(-\frac{3+1}{2}\) \(-2\)
    4 \(\frac{4}{2}\) \(2\)
    5 \(-\frac{5+1}{2}\) \(-3\)
    \(\vdots\) \(\vdots\) \(\vdots\)

    Claim: \(f(n)\) is a bijection from the naturals to the integers.

    In order to verify that \(f\) is a bijection from the natural numbers to the integers, we need to verify that \(f\) is a function from the natural numbers to the integers, that is is an injection, and that it is a surjection.

    We know that \(f\) is a function from the natural numbers to the integers because whenever \(n\) is a natural number, \(\frac{n}{2}\) and \(-\frac{n+1}{2}\) are both integers.

    We know that \(f\) is injective because, whenever \(n_1\) and \(n_2\) are natural numbers we have the following: (a) the only way for it to be the case that \(\frac{n_1}{2} = \frac{n_2}{2}\) is for it to be the case that \(n_1=n_2\), and (b) the only way for it to be the case that \(-\frac{n_1+1}{2} = -\frac{n_2+1}{2}\) is for it to be the case that \(n_1=n_2\).

    We know that \(f\) is surjective because: (a) each non-negative integer is of the form \(\frac{n}{2}\) for some even natural number \(n\), and (b) each negative integer is of the form \(-\frac{n+1}{2}\) for some odd natural number \(n\).

    End of proof.

    Also the same cardinality as \(\mathbb{N}\): the even natural numbers. Also: the multiples of 7. Exercise: prove this.

The Rational Numbers

A rational number is a number equal to \(\frac{a}{b}\), where \(a\) is an integer, and \(b\) is an integer distinct from zero. (For instance, \(\frac{17}{3}\) is a rational number, and so is \(4=\frac{4}{1}\).)

Here is a surprising fact: the cardinality of the set of rational numbers \(\mathbb{Q}\) is the same as the cardinality of the set of natural numbers.

The easiest way to see this is by building a matrix, such as the following:

Every (non-negative) rational number appears on some cell of our matrix. So we can show that there is a bijection between the natural numbers and the (non-negative) rational numbers by assigning a different natural number to each cell.

There are many different ways of doing this. One is to follow the route below:

As you advance through the red path, you assign natural numbers to (non-negative) rational numbers: \[0 \rightarrow \frac{0}{1}\] \[1 \rightarrow \frac{1}{1}\] \[2 \rightarrow \frac{0}{2}\] \[3 \rightarrow \frac{0}{3}\] \[4 \rightarrow \frac{1}{2}\] \[\vdots\]

Actually, a slight adjustment is needed to correct for the fact that rational numbers appear on the matrix multiple times under different names (\(0\), for example, appears as “\(\frac{0}{1}\)”, “\(frac{0}{2}”, “\frac{0}{3}”, etc.). We can address the issue by following the same route, but skipping any cells with rational numbers that have already been counted. So the assignment we get is actually the following: \[0 \rightarrow \frac{0}{1} = 0\] \[1 \rightarrow \frac{1}{1} = 1\] \[2 \rightarrow \frac{1}{2}\] \[3 \rightarrow \frac{2}{1} = 2\] \[4 \rightarrow \frac{3}{1} = 3\] \[\vdots\]

This assignment immediately yields a bijection \(f\) from the natural numbers to the (non-negative) rational numbers: \[f(0) = 0\] \[f(1) = 1\] \[f(2) =1/2\] \[f(3) = 2\] \[f(4) = 3\] \[\vdots\]

This construction gives you a bijection from the naturals to the non-negative rationals. Exercise: show that there is a bijection from the naturals to the rationals (i.e., the set of both negative and non-negative rationals).


The Real Numbers

Lots of infinite sets have the same cardinality as the natural numbers. Do all infinite sets have the same cardinality as the natural numbers? No! The first result to this effect was proved by Cantor, and it is the result that started the whole field of study of trans-finite cardinalities (as they are called).

The result is that the cardinality of the real numbers is larger than the cardinality of the natural numbers.

Let’s prove it!

    Step 1: The Cardinality of the Naturals is Less Than or Equal to the Cardinality of [0,1]

    This part is pretty easy. Consider the function \(f=\frac{1}{n+1}\). Claim: that is a an injection from \(\mathbb{N}\) to \([0,1]\). That should be pretty easy to convince yourself of; just note that if \(n_1 \neq n_2\), then \(\frac{1}{n_1+1}\neq \frac{1}{n_2+1}\).


    Step 2: The Cardinality of the Naturals is Not Equal to the Cardinality of [0,1]

    This is the ingenious part. The proof is by reductio ad absurdum; that is, we assume something, show that it leads to contradiction, and conclude the negation of the thing that we assumed. In this case, we assume that \(|\mathbb{N}| = |[0,1]|\), derive a contradiction, and conclude that \(|\mathbb{N}|\neq |[0,1]|\).

    So: suppose that \(|\mathbb{N}| = |[0,1]|\). Then there is a bijection \(f\) from \(\mathbb{N}\) to ([0,1]). If \(f\) exists, it can be used to make a list of all real numbers between 0 and 1:

    • The first member of our list is \(f(0)\), which will be some particular real number ; say, \(0.a_1a_2a_3\ldots\) (where \(a_1\), \(a_2\), \(a_3\),… are digits between “0” and “9”).
    • The second member of our list is \(f(1)\), which is again some particular real number; say, \(0.b_1b_2b_3\ldots\) (where \(b_1\), \(b_2\), \(b_3\),… are digits between “0” and “9”).
    • The third member of our list is \(f(2)\), which is again some particular real number; say, \(0.c_1c_2c_3\ldots\) (where \(c_1\), \(c_2\), \(c_3\),… are digits between “0” and “9”).
    • And so forth.

    We can represent it using this table, with the \(n\)th row representing \(f(n)\).

    Now consider the digits on the table that have been marked in red. (Don’t forget that the diagonal is supposed to go infinitely far…)

    We can use these digits to build the “diagonal” real number \(D\) , as follows: \[D=0.a_1b_2c_3d_4e_5\ldots\]

    (Because each of \(a_1,b_2,c_3,d_4,e_5,\ldots\) is a digit between “0” and “9”, \(D\) must be a real number between 0 and 1.)

    The thing about \(D\) is that it has an evil twin: the number \(D^*\).

    \(D^*\) is simply the result of applying the “evil” function \(e\) to each of the digits in \(D\): \[D^*=0.e(a_1)e(b_2)e(c_3)e(d_4)e(e_5)\ldots\]where \[e(x)=7\text{ if }x\neq7\] \[e(x)=3\text{ if }x=7\]

    Suppose, for instance, that our list of reals were to be filled out as follows:

    Then \(D = 0.7317\ldots\), and \(D^* = 0.3773\ldots\).

    Like \(D\), \(D^\star\) is simply a real number between 0 and 1. What is interesting about \(D^\star\) is that we have a way of showing that it must be different from each of the real numbers that make up the rows of our table. It must be different from the number on the first row because that number has \(a_1\) as its first digit and \(D^\star\) has \(e(a_1)\) (which is different from \(a_1\)); it must be different from the number on the second row because that number has \(b_2\) as its second digit and \(D^*\) has \(e(b_2)\) (which is different from \(b_2\)); and so forth, for every row of our table.

    (In our example, you can tell that \(D^*\) is different from the number on the first row of the table because that number has “7” as its first digit and \(D^*\) has “3”; you can tell that \(D^*\) is different from the number on the second row because that number has “3” as its second digit and \(D^*\) has “7”; and so forth.)

    So we know that \(D^*\) is a real number between 0 and 1 which is not on our table. But that contradicts the assumption we made earlier: that our table includes every real number between 0 and 1. So \(|\mathbb{N}| = |[0,1]|\) cannot be true, after all. So not-(\(|\mathbb{N}| = |[0,1]|/)) must be true, which is what we wanted to prove.

      A Small Complication

      There is a small worry you might have, that has to do with the fact that we are dealing with names of real numbers, rather than real numbers, in the above proof.

      Some real numbers have two names in decimal notation; for instance, \(\frac{1}{2}\), which has the names “0.4999…”, and “0.5000…” in decimal notation.

      So, you might worry, the fact that a certain name of a real number isn’t on our list — that’s what the diagonal argument shows — doesn’t mean that that number isn’t on our list somewhere, under a different name.

      Thankfully, this is not a big problem for our proof. The only real numbers that have two names in decimal notation are numbers with a name that ends in all “0”s, and a name that ends in all “9”s. And the formula given above for finding the evil number finds a number with a name that has neither “0”s nor “9”s in it. So that number has only one name in decimal notation. So if that name isn”t on the list, that number is not represented under any name on the list.

From step 1 and step 2, we can conclude that the cardinality of the naturals is less than the cardinality of [0,1]. Amazing!

    Step 3: The Cardinality of [0,1] Equals the Cardinality of the Reals

    To show this, we need to show that there is a bijection from \([0,1]\) to \(\mathbb{R}\) (i.e., the set of real numbers). That is actually a bit trickier than you might think. Here’s a way to do it.

      Step 3a: The Cardinality of (0,1) Equals the Cardinality of the Reals

        Step 3a-i: The Cardinality of (-π/2, π/2) Equals the Cardinality of the Reals

        Consider \(f(x)=\text{tan}(x)\). That is a bijection from \((-\frac{\pi}{2}, \frac{\pi}{2})\) to \(\mathbb{R}\). The easiest way to see this is to look at the graph of \(\text{tan}(x)\). Convince yourself!


        Step 3a-ii: The Cardinality of (0,1) Equals the Cardinality of (-π/2, π/2)

        Consider \(f(x) = \pi(x - \frac{1}{2})\). That’s a bijection from \((0,1)\) to \((-\pi/2,\pi/2)\). Convince yourself!

      From steps 3a-i and 3a-ii we can conclude that the cardinality of \((0,1)\) equals the cardinality of \(\mathbb{R}\); we are using the transitivity of bijections to conclude this.

      We can, in fact, state explicitly one such bijection; it is \(\text{tan}\circ f\), where \(f\) is the function mentioned in step 3a-ii.


      Step 3b: The Cardinality of [0,1] Equals the Cardinality of (0,1)

      This is the trickiest part of Step 3. Here’s how to do it.

      We define a bijection from \([0,1]\) to \((0,1)\).Take some subset of \([0,1]\) with the cardinality of the natural numbers — say, the set of all numbers \(\frac{1}{n+1}\) for all \(n\in\mathbb{N}\). Call that set \(S\).

      Let every number in \([0,1]\) be mapped to itself except for \(0\), \(1\), and the members of \(S\).

      Let \(0\) be mapped to \(\frac{1}{2}\). Let \(1\) be mapped to \(\frac{1}{3}\).

      Let each \(frac{1}{n+1}\in S\) be mapped to \(frac{1}{n+3}\).

      In summary: \(f\) is the function from \([0,1]\) to \((0,1)\) such that: \[f(0)=\frac{1}{2}\] \[f(1)=\frac{1}{3}\] \[\text{for all }\frac{1}{n+1}\in S\text{, }f(\frac{1}{n+1})=\frac{1}{n+3}\text{, and }\] \[f(x)=x\text{ otherwise.}\]

      \(f\) is a bijection. (Convince yourself this is so!)

    Putting this all together, we get: \(|[0,1]|=|\mathbb{R}|\). Success!

Putting that together with step 2, we get: \(|\mathbb{N}|\le|\mathbb{R}|\). Double success!

Putting that together with step 1, we get \(|\mathbb{N}|<|\mathbb{R}|\). Hooray!


A Helpful Result

The technique we used in the last part of Step 3, above, can be generalised to a useful result.

A set \(C\) is countable iff \(|C|\le |\mathbb{N}|\). Here is the result.

If \(|A|\ge|\mathbb{N}|\), then, for any countable \(C\), \(|A\cup C|=|A|\).

Here is the proof:

Take a subset of \(A\) with the cardinality of \(\mathbb{N}\).1 Call that subset “\(A^\mathbb{N}\)”, and call its elements “\(a_0, a_1, a_2,\ldots\)”. First we show that

Lemma: \(|A^\mathbb{N}\cup C| = |A^\mathbb{N}|\) (and, hence, equals \(|\mathbb{N}|\)).

How’s that? Take all elements of \(C\) that aren’t also in \(A^\mathbb{N}\) — call the set of them “\(C^-\)”. At most, \(|C^-|=|\mathbb{N}|\). Suppose it does. Call the elements of \(C^-\) “\(c_0, c_1, c_2,\ldots\)”

Let \(f\) be the function that maps \(a_0, a_1, a_2,\ldots \in A^\mathbb{N}\cup C\) to \(a_0, a_2, a_4,\ldots\in A^\mathbb{N}\), and maps \(c_0, c_1, c_2,\ldots\in A^\mathbb{N}\cup C\) to \(a_1, a_3, a_5,\ldots\in A^\mathbb{N}\).

\(f\) is a bijection from \(A^\mathbb{N}\cup C\) to \(A^\mathbb{N}\). (Convince yourself that this is so!)

If \(|C|<|\mathbb{N}\), then it is even easier to show that there is a bijection from \(A^\mathbb{N}\cup C\) to \(A^\mathbb{N}\). I leave this as a fun diversion for the reader.

Now, here’s a bijection from \(A\cup C\) to \(A\). Map everything in \(A\cup C\) to itself except for the members of \(A^\mathbb{N}\cup C\). Map those elements to \(A\mathbb{N}\), via \(f\).

And so: \(|A^\mathbb{N}\cup C|=|A|\)


R, R2, …

So the cardinality of the reals is greater than the cardinality of the naturals. And you can see, from the proof, that any interval of real numbers has the same cardinality as the set of all the reals, and hence a greater cardinality than the naturals. Crazy!

What about \(\mathbb{R^{2}}\) — that is, the set of all ordered pairs of real numbers? That too has the cardinality of the real numbers! Here is a proof.

    Step 1: |[0,1]d|=|[0,1]d × [0,1]d|

    Let \([0,1]^d\) be the set of all names of real numbers between 0 and 1 in decimal notation. We prove that that set has the same cardinality as \([0,1]^d\times [0,1]^d\) — i.e., the set of all ordered pairs of names of real numbers between 0 and 1.

    The following is a bijection: \[ f(0.a_1a_2a_3a_4\ldots) = \langle 0.a_1a_3a_5a_7\ldots,0.a_2a_4a_6a_8\ldots\rangle ]


    Step 2: (|0,1| = |0,1d|)

    Let \(D\) be the set of real numbers in \([0,1]\) with duplicate decimal expansions, and let \(D^d\) be all names in decimal notation of numbers in \(D\).

    The crucial observation is that there are two names in \(D^d\) for each member of \(D\), one of which ends in zeroes, the other in nines. Since only a rational number can have a decimal expansion ending in all zeroes, or ending in all nines, this means that every number in \(D\) is a rational number, and therefore that \(D\) is countable. For each number in \(D\), \(D^d\) contains two names, and is thus also countable.

    Let \([0,1]^0\) be the result of removing from \([0,1]^d\) every expansion ending in all nines. Since there are no longer any duplicate expansions in \([0,1]^0\), and since every member of \([0,1]\) still has an expansion in \([0,1]^0\), we know that \[|[0,1]^0| = |[0,1]|\].

    Now, \(|[0,1]^0|\ge \mathbb{N}\) (there is an injection from \(\mathbb{N})\) to \(|[0,1]^0\) via \(f=\frac{1}{n+1}\) and via the function from a real number a name for that number in decimal notation.) And \(D^d\) is countable. So, by the handy result above, \(|[0,1]^0| = |[0,1]^0\cup D^d|\). And \([0,1]^0\cup D^d\) just is \([0,1]^d\). So \[|[0,1]^0| = |[0,1]^d|\] From which it follows that \[|[0,1]|=|[0,1]^d|\].

    As a corollary, \(|[0,1] \times [0,1]| = |[0,1]^d \times [0,1]^d|\).

Putting steps 1 and 2 together, we get \[|[0,1]|=|[0,1]\times [0,1]\]

We already know that \(|\mathbb{R}|=|[0,1]|\), and, hence, that \(|[0,1]\times [0,1]| = |\mathbb{R}\times \mathbb{R}|\). So, from above, we get \[|\mathbb{R}|=|\mathbb{R}^2|\]

We can go further. A similar construction to the one above shows that \(|[0,1]| = |[0,1]\times [0,1] \times [0,1]|\), and another shows that \(|[0,1]| = |[0,1]\times [0,1] \times [0,1]|\times [0,1]\), and so on, and, hence that, for all \(n\in\mathbb{N}\), \[|\mathbb{R}|=|\mathbb{R}^n|\]

And, in fact, \[|\mathbb{R}|=|\mathbb{R}^\mathbb{N}|\] (though you can’t prove it using the method above).

Whoa!


Cantor’s Theorem

So now we have two infinite cardinalities: the cardinality of the natural numbers, and the cardinality of the real numbers. And we’ve seen that it is not easy to find a set any bigger than the reals. Is that it? Have we got as big as it gets?

No. We haven’t. And Cantor showed that we can never get as big as it gets. For every set, there is a bigger set. There are infinite number of infinite cardinalities!

The result I will now prove is called Cantor’s Theorem. It says, to put it roughly, that

A set always has more subsets than it has members.

To put it more formally: a power set \(\mathcal{P}(A)\) is the set of all subsets of \(A\).2 Cantor theorem states that \[|\mathcal{P}(A)|>|A|\]

    The Proof

    We have to prove that

    1. \(|A| \leq |\mathcal{P}(A)|\), and
    2. \(|A| \neq |\mathcal{P}(A)|\)

    The first of these is relatively easy to prove, since the function \(f(x) = \{x\}\) is an injection from \(A\) to \(\mathcal{P}(A)\). So what we really need to prove is the second statement: \(|A| \neq |\mathcal{P}(A)|\).

    Our proof will proceed by reductio ad absurdum. So we will assume the negation of \(|A| \neq |\mathcal{P}(A)|\): \[|A| = |\mathcal{P}(A)|\] and use this assumption to prove a contradiction.

    In order for our assumption to be true, there must be a bijection \(f\) from \(A\) to \(\mathcal{P}(A) \).

    Notice that, for each \(a\) in \(A\), we can ask the following question:

    \[a \in f(a)\text{?}\]

    (If, for example, \(f\) maps \(a\) to \(\{\}\), then not-(\(a \in f(a)\)). If, on the other hand, \(f\) maps \(a\) to \(\{a\}\), then \(a \in f(a)\).)

    With this in mind, let us define \(D\) to be the subset of \(A\) such that

    for any \(x\), \(x \in D\) if and only if not-(\(x \in f(x)\))

    We now note the following two things:

    1. \(D\) is a subset of \(A\), and therefore \(D \in \mathcal{P}(A)\).
    2. \(f\), being a bijection, is a surjection from \(A\) to \(\mathcal{P}(A)\).

    From these two things together it follows that there must be some \(d \in A\) such that \(f(d) = D\).

    Question: \(d \in D\)?

    By the definition of \(D\), \(d \in D\) if and only if not-(\(d \in f(d)\)).

    Since \(f(d) = D\), this can be restated as follows:

    \(d \in D\) if and only if not-(\(d \in D\)).

    But that is impossible! It says that something is the case if and only if it is not the case.

    We have arrived at our contradiction. So our assumption must be false. In other words: it cannot be the case that \(|A| = |\mathcal{P}(A)|\).

    So we have shown that \(|A| < |\mathcal{P}(A)|\). We have shown that, regardless of how many things there are in \(A\), there must be more in \(\mathcal{P}(A)). Amazing!
